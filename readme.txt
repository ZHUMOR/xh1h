基于内核学习的打地鼠小游戏
本项目是通过线程之间的互相调度，邮箱之间的相互通信进行完成的。    
实现了一款打地鼠的小游戏，用户通过开机启动，观察左边LED灯环的颜色，0号位LED灯亮绿色，代表游戏开始，随后会在灯环中，随机依次亮起三个灯，用户需要凭借记忆力，按顺序记得每个灯的位置。随后当第三个灯亮灭后，中间位置的蓝灯即是用户所需要操控的指示灯，
通过下键是用来切换灯圈的，顺序是：外圈—内圈—中心圈。
通过左键和右键，能够实现向左移动和向右移动
通过上键，是确认按键和提交按键
当依次选择三个位置时，第四次按下上键即是确认按钮，提交你所选择的答案。

如果回答准确，LCD显示屏会显示，成功通关的字样，如果失败，蜂鸣器会发出提示音，便需要重新回答。
LCD显示部分：
显示了用户选择的位置信息，即第一次选择的点位，第二次选择的点位，第三次选择的点位。方便用户去查看选择的信息。

实现过程：
通过了四个线程进行设计
一线程：负责管理LED灯，和处理LED灯的位置信息
二线程：负责处理按键按下后的信息
三线程：负责控制LED灯移动
四线程：负责处理用户答案和标准答案的匹配关系
通过两个邮箱处理线程与线程之间的通信
一号邮箱负责按键按下后传递信息给各个线程
二号邮箱负责处理LED灯的移动信息和选择答案的处理
